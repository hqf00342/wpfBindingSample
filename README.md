# wpf ドラッグ移動サンプル

MVVM Bindingサンプル。

- WPF / .NET 6.0 
- MVVMライブラリ未利用。MvvmUtilsフォルダに単純なものを用意。
- `RectInfo`というデータクラスでRectangleを管理
- Canvas上のUIElementをドラッグ移動させるとDataGridに反映。
- 逆方向も反映
